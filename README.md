# Professional Portfolio

This is a web application that showcases my professional experience, skills, and projects. The goal of this portfolio is to give potential clients and employers an insight into my work and to demonstrate my capabilities as a actively learning web developer.

## Technologies Used ğŸ› ï¸

This portfolio is created using HTML, CSS, and JavaScript. It utilizes modern web development tools and frameworks, such as Bootstrap and FontAwesome, to provide an appealing and functional design.

- [React](https://reactjs.org/)
- [graphql](https://graphql.org/)
- [apollo-boost](https://www.apollographql.com/docs/react/get-started/)
- [baseui](https://github.com/uber/baseweb)
- [react-reveal](https://www.react-reveal.com/)
- [styled-components](https://styled-components.com/)

## Web Pages

The portfolio consists of several web pages that contain information on my experience, skills, and projects. These pages include:

- About Me: This page provides a brief introduction to who I am and what I do.

- Experience: This page provides details on my previous work experience, highlighting my skills and accomplishments in each role.

- Skills: This page lists my areas of expertise, including technical skills as well as soft skills like communication and leadership.

- Projects: This page showcases some of my recent projects and provides links to live demos or the source code.

## Set Up and Installation

To use this portfolio, simply download the repository or clone it to your local machine. Open the index.html file in your web browser to view the application.

## Future Plans

I plan to continually update this portfolio with new projects and experiences as I progress in my career. I am also considering implementing additional functionality, such as a contact form or blog section on my website.
